## OG Android Template

The OG Android Template provides all the boilerplate, startercode, and dependencies needed for a Kotlin-first, MVI/MVVM Android application.

## Dependencies Included

 - Kotlin
 - AndroidX
 - Dagger
 - Espresso
 - Glide
 - OkHttp
 - Retrofit
 - Room
 - RoboElectric
 - Coroutines
 - RxJava
 - Timber
 - Moshi
